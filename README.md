# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/joita-paul-portfolio.git
```

Move into the project:

```bash
cd joita-paul-portfolio
```

Install dependencies:

```bash
npm install
```

Create environment variables:

```bash
cp .env.example .env
```

Add your Gemini API key:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run development server:

```bash
npm run dev
```

Build production:

```bash
npm run build
```

Start production server:

```bash
npm run start
```
